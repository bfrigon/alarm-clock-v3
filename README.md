alarm-clock (V3)
================

Project page [here][link-project]

Specs
-----

- Arduino based design using the ATMEGA2560 MCU.
- WI-FI connectivity
- Plays MP3/WAV files from a SD card.
- Large RGB seven segment display
- 16x2 character LCD display
- Night lamp
- Touch keypad
- 24h battery backup
